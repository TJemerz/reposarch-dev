# Loops 🔄️

1. **for** → loops through a block of code a number of times
1. **for/in** → loops through the properties of an object
1. **for/of** → loops through the values of an iterable object
1. **forEach** → calls a function (a callback function) once for each array element.
1. **while** → loops through a block of code while a specified condition is true
1. **do/while** → also loops through a block of code while a specified condition is true
